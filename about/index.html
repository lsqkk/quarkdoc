关于本站

欢迎来到 QuarkDoc 文档中心。

这里是站主 蓝色奇夸克 的个人博客 夸克博客 旗下的一个文档项目。

项目初衷

本仓库旨在系统地整理、沉淀和分享我在学习、工作与兴趣探索中产生的各类内容。它不同于即时的博客文章，更侧重于构建一个结构清晰、便于查阅的知识库。

主要内容

这里不定期更新与维护的内容可能包括（但不限于）：

  • 资料整理：关于特定技术栈或领域的系统性笔记与教程。
  • 博客相关：对博客中长文或系列文章的延伸阅读、补充材料或更新日志。
  • 开源项目文档：我所维护或贡献的开源项目的详细使用说明、开发指南和API参考。
  • 其他思考：一些尚未成文或适合以文档形式呈现的想法与总结。

技术构建

  • 生成工具：本站使用 MkDocs 静态网站生成器构建。
  • 主题样式：采用 MkDocs 自带的 readthedocs 主题，并进行了定制化修改。
  • 部署发布：通过 MkDocs 自动构建并推送至 GitHub Pages。
